Nesse projeto, já estão definidas duas classes principais:

Banco

Conta

A classe Banco possui uma relação de associação 1..n com a classe Conta, representando que um banco pode possuir várias contas.
Além disso, já existe uma interface de console parcialmente implementada. Atualmente, o menu possui apenas as funcionalidades de:

Inserir conta

Listar contas

No entanto, ainda faltam quatro operações essenciais que você deverá desenvolver:

Sacar

Depositar

Aumentar limite

Diminuir limite

Sua tarefa
Acesse o repositório original e faça um fork em uma conta sua do GitHub.

Implemente as quatro funcionalidades faltantes na interface console, utilizando corretamente os métodos das classes Banco e Conta.

Após finalizar, publique o código atualizado no seu repositório.

Envie o link do seu repositório modificado na Plataforma A.
